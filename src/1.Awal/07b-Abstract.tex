\textit{PT Pratama Nusantara Sakti is a sugarcane plantation company that manages
a large number of contract workers, yet still relies on a Microsoft Access-based
application that is prone to errors, data duplication, and administrative fraud.
This internship aimed to design and develop HR Safety Campus, a web-based contract
worker administration system built using the Laravel framework with a
Model-View-Controller (MVC) architecture and the Waterfall development method.
The system implements Role-Based Access Control (RBAC) to restrict user
privileges, a responsive interface based on Tailwind CSS, and a data
deduplication mechanism during the migration of approximately 4,000 worker
records from the legacy system to a MySQL database. The internship produced six
main features, namely worker registration, transport claims, exit permits, rate
management, employee approval, and user management, which were successfully
integrated and tested. With this system, the worker administration process
becomes faster, more accurate, and more centralized, while closing the data
manipulation loopholes that previously harmed the company.}
